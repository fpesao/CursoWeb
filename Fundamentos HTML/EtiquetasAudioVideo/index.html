autoplay

Reproduce el video o audio apenas se abre la web, sin necesidad de dar click

loop

Reproduce el video o audio nuevamente cada vez que termine

preload

Almacena en buffer los archivos una vez que se hayan cargado, se usa cuando trabajamos con archivos muy pesados

  • none: No almacenar nuestros archivos
  • auto: Almacena temporalmnete el archivo
